砷化鎵晶圓代工大廠穩懋半導體第三季營收及毛利率均呈季成長，單季每股盈餘（EPS）3.67 元，毛利率回升至 38.8%，優於預期，前三季每股盈餘 8.71 元；穩懋預估第四季營收季增中個位數百分比，毛利率在 37%~39% 間第三季相當。

穩懋今天舉行線上法人說明會，公佈第三季財報，第三季單季營收新臺幣67.61億元，較前一季增加9%，較去年同期增加3%。第三季在產品組合及產能利用率雙重影響下，營業毛利率回升到38.8%，較前一季增加3.1個百分點；營業淨利率26.8%，較前一季增加4.1個百分點。

穩懋總管理服務處總經理陳舜平表示，3D感測為主的相關產品第三季成長幅度最大，5G PA、基礎建設PA均較上季個位數成長，第三季產能利用率約到95%。

展望第四季，陳舜平表示，預計第四季營收將較前一季成長中個位數（mid-single digit）百分比，毛利率預估約37%~39%（high-thirties），第四季產品組合相較第三季變化不大。

產能利用率部分，陳舜平表示，由於第三季已到95%，預估第四季還會再攀升，有助於毛利率表現，但整體來看還是認為落在high-thirties的機率較高。

針對法人關心3D感測晶片市場競爭態勢，陳舜平指出，一開始以來穩懋直接客戶就不是終端客戶唯一供應來源，美國當地也有其他供應商，競爭一直都有，但穩懋維持相當優勢，截至目前每代產品都取得最大比例佔比。

至於中國手機需求，陳舜平說，從客戶訂單來看，第四季營收還是有持續成長，沒有看到轉弱；他強調，PA屬非常客製化晶片，基本上不太會有同一個晶片用好幾年，因為每環節對庫存都有控管。穩懋5G的PA客戶群已涵蓋一線品牌，包括美系及非美系客戶。

中低軌道衛星部分，穩懋發言人曾經洲表示，2~3年前就佈局並小量出貨，從數量角度看，不容易有爆發性成長，但今年的確見到較顯著成長，也嘗試不同技術提供更具效能的產品，目前逐季都有穩定營收貢獻，單看中低軌道衛星產品，目前營收佔比仍在低個位數。

穩懋位於南部科學園區高雄園區路竹新廠，由於總面積大過桃園三廠總和，新產能會分階段逐步開出布建，預計最快2024~2025年才會貢獻營收。
