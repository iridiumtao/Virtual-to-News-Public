### Structure 專案結構說明
```
├── assets 資產，放置圖片、文字檔等「素材類」檔案
│   └── news_web_list.txt
├── docs
│   └── docs.txt
├── LICENSE 授權條款，由 GitHub 生成
├── main.py 整隻程式的執行入口，必須有 if __name__ == "__main__"
├── references 放可以參考的、以前寫的code
├── README.md 本說明文件
├── scripts 可以直接 terminal 執行的輔助程式
│   └── web_crawl.py 
└── virtual-to-news 相當於 /src 資料夾，放置專案所需的程式檔
    ├── model.py
    └── test 測試用程式置於此
        └── test.py
```
